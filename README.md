# About Device

![Nokia 8](https://drop.ndtv.com/TECH/product_database/images/8162017114310PM_635_nokia_8_back_silver.jpeg)

Announced on 16 August 2017, the Nokia 8 (codenamed _"NB1"_) is a flagship smartphone from HMD Global ( Nokia ). The NB1 now supports TWRP!

### Specifications

Component Type | Details
-------:|:-------------------------
Chipset | Qualcomm MSM8998 Snapdragon 835
CPU     | Octa-core (4x2.5 GHz Kryo & 4x1.8 GHz Kryo)
GPU     | Adreno 540
Memory  | 4GB RAM
Shipped Android Version | 7.1.1
Storage | 64GB
Battery | Non-removable Li-Ion 3090 mAh battery
Display | 5.3 inches, 77.4 cm2 (~69.4% screen-to-body ratio)
Rear Camera | Dual: 13 MP, f/2.0, 1.12µm, PDAF & laser AF, OIS + 13 MP B/W, f/2.0, 1.12µm, PDAF & laser AF, Zeiss optics, dual-LED dual-tone flash,HDR, panorama,2160p@30fps, 1080p@30fps
Front Camera | 	13 MP, f/2.0, 1.12µm, PDAF,2160p
